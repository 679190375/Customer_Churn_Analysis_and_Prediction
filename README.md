## 📊 Customer Churn Analysis and Prediction

This project analyzes customer churn in a telecommunications company 
and builds machine learning models to identify at-risk customers.

### 🔍 What's covered:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Customer segmentation by tenure and charges
- Churn prediction using Logistic Regression, Decision Tree, Random Forest, and XGBoost
- Model evaluation with ROC curves and AUC scores
- Business recommendations to reduce churn and save revenue

### 🏆 Best Model:
Logistic Regression — Accuracy: 82%, AUC: 0.86

### 💰 Key Finding:
Monthly revenue at risk: $139,130 — retaining 30% of churners 
saves over $500,000 annually.

### 🛠️ Tools Used:
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Google Colab
